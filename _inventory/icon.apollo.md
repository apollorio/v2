## Notes

- All icons use RemixIcon classes (`ri-*`) unless specified otherwise.
- Custom Apollo icons have multiple class options for flexibility.
- Forbidden: Using bell icons for notifications; use notif/signal-tower variants instead.
- Ensure icons are loaded via the Apollo CDN for consistency.
