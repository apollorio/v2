Apollo Design System

Universal
Style Toolkit

Catálogo visual de componentes do Apollo. Cards, formulários, selects, notificações — tudo usando as variáveis do CDN como base única.

Dica Importante — dvh, não vh!
Sempre use 100dvh ao invés de 100vh para layouts full-height. No Safari iOS, 100vh inclui a altura da barra de URL, causando overflow e scroll indesejado. dvh (Dynamic Viewport Height) se adapta automaticamente quando a barra some/aparece.

Regras:
✓ Modais: height: 100dvh
✓ Overlays: min-height: 100dvh
✓ Formulários full-screen: height: 100dvh
✗ Nunca: height: 100vh em elementos fixos/modais
Como usar
1. Carregue o CDN Apollo: <script src="https://cdn.apollo.rio.br/v1.0.0/core.js">
2. Importe o stylesheet: <link rel="stylesheet" href="universal-style.css">
3. Use as classes e variáveis CSS diretamente no HTML.
4. O CDN injeta automaticamente: reset, tipografia, cores, dark mode e utilitários.

Componente — Formulários

Form Base

Inputs com floating labels, borda inferior e animação no foco. O modal desliza da direita com overlay blur. Clique no botão para testar.

Form Tips
• O modal usa height: 100dvh para nunca estourar no mobile.
• O submit-btn usa margin-top: auto para colar no fundo do modal via flexbox.
• Use placeholder=" " (espaço) nos inputs para ativar :not(:placeholder-shown).
• Overlays devem ter backdrop-filter: blur() + opacity transition.
